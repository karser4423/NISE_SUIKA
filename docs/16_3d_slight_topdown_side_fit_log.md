# 3D版左右見切れ防止のための軽い俯瞰調整記録

## 2026-06-11

### 変更内容

- `index_3d.html` の縦長時カメラを少し上から見下ろす角度に調整。
- 縦長時のカメラ位置を少し高くし、注視点をわずかに奥側へ戻した。
- ステージ手前側の横幅を `5.8` から `5.2` に微調整。
- 下部70%にステージ、上部30%にHUDを置く構成は維持。

### 変更理由

- 下部70%レイアウト適用後、ステージ手前側の左右端が再び画面外へ見切れていたため。
- パースの角度を少し上からの視点へ寄せ、横方向の膨らみを抑えるため。

### 確認内容

- `http://127.0.0.1:5500/index_3d.html` を縦長ビューポートで表示し、ステージ左右端が画面内に収まることを確認。

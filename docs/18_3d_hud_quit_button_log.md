# 3D版HUD重なり解消と中断ボタン追加記録

## 2026-06-11

### 変更内容

- `MODE / SHOT / TIME` の表示を、注文票とは別の絶対配置ではなくHUD内の下段に移動。
- HUDを上段のスコア・注文票・コイン、下段のモード・射出数・タイム・やめるボタンという構成に変更。
- プレイ中だけ表示される `やめる` ボタンを追加。
- `やめる` ボタン押下時に結果画面へ遷移し、結果タイトルを `中断`、説明を `プレイを中断しました。` と表示するようにした。
- トーストメッセージの表示位置をHUD下へずらし、HUD表示と重なりにくくした。

### 変更理由

- 画面サイズによって `MODE / SHOT / TIME` の表示が注文票に重なることがあったため。
- 各モードのプレイ中に、ユーザーが任意に中断してモード選択やリトライへ戻れる導線が必要なため。

### 確認内容

- 縦長ビューポートで `ORDER TICKET` と `MODE / SHOT / TIME` が重ならないことを確認。
- プレイ中に `やめる` ボタンが表示され、押下すると結果画面の `中断` 表示へ遷移することを確認。

# 調整パラメーター初期値変更記録

## 2026-06-10

### 変更内容

- HTMLプロトタイプの重力スライダー初期値を `145` から `85` に変更。
- HTMLプロトタイプの摩擦スライダー初期値を `0.925` から `0.995` に変更。
- HTMLプロトタイプの射出速度スライダー初期値を `390` から `620` に変更。
- 企画メモの調整パラメーター欄に、現時点の推奨初期値として `85 / 0.995 / 620` を追記。

### 変更理由

- 試行錯誤の結果、重力 `85`、摩擦 `0.995`、射出速度 `620` の組み合わせが現時点で気持ちよい挙動に近いため。
- プロトタイプを開いた直後から、確認済みの操作感で遊べるようにするため。

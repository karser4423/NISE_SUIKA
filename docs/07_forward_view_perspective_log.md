# 手前から奥を見る視点への変更記録

## 2026-06-10

### 変更内容

- ステージ奥側の絞り量を幅の `13%` から `34%` に変更。
- 奥へ進んだときのオブジェクト縮小率を強め、奥側では手前よりかなり小さく見えるように変更。
- テーブル上に奥行き方向の横ラインを追加し、手前から奥へ向かう距離感を分かりやすくした。
- 奥の行き止まりを低い線ではなく、厚みのある壁として描画するように変更。
- 描画上のパースと左右壁の衝突判定がずれないよう、同じ奥側絞り量を維持。

### 変更理由

- 前回の調整ではステージがほぼ真上から見ているように見え、手前から奥方向を見ている印象が弱かったため。
- 射出方向が「手前から奥」であることを視覚的に分かりやすくするため。

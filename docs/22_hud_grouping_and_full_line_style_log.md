# HUD整理と満杯ライン見た目改善

## 変更日

2026-06-13

## 現状調査

- HUDは `score`、`order ticket`、`coin`、`mode`、`shot`、`time`、`next`、`evolution` が個別に並び、情報の種類が混在していた。
- 満杯ラインは `THREE.Line` で赤系の細い線として描画しており、ステージの見た目に対して目立ち方が強かった。

## 変更内容

- `score`、`coin`、`shot`、`time` を数字情報として同じ行にまとめた。
- `order ticket`、`next`、`evolution` をアイコン情報として同じまとまりに整理した。
- 通常のスマートフォン幅では `order ticket` と `next` を横並びにし、かなり狭い画面だけ縦積みにした。
- `mode` と `やめる` を操作情報として同じ行にまとめた。
- HUDの表示順を、操作情報、数字情報、進化情報、注文・次オブジェクト情報の順に変更した。
- 満杯ラインを赤い `THREE.Line` から、ステージ面に置く少し太い白い帯状の `THREE.Mesh` に変更した。
- 開始時などのメッセージ表示位置をHUD下へ下げ、進化段階表示と重なりにくくした。

## 理由

- 数字系、アイコン系、操作系で視線の目的を分けることで、HUDの情報量が増えても読み取りやすくするため。
- 満杯ラインを白い帯にすることで、ステージのグリッドやドリンクと馴染ませつつ判定位置を確認できるようにするため。

## 確認項目

- HUDが数字情報、アイコン情報、操作情報の順にまとまって表示されること。
- SCORE、COIN、SHOT、TIMEが同じ行で確認できること。
- ORDER TICKET、NEXT、EVOLUTIONが同じ情報群として確認できること。
- MODEとやめるボタンが操作情報としてまとまっていること。
- 満杯ラインがステージ上で白い太めの線として表示されること。

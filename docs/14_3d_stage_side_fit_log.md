# 3D版ステージ左右見切れ調整記録

## 2026-06-11

### 変更内容

- `index_3d.html` のゲーム領域を、固定最大サイズではなくビューポート全体に広げるように変更。
- ステージ手前側のZ位置を `4.9` から `2.4` に変更し、カメラに近すぎて左右へ膨らみすぎる状態を緩和。
- ステージ手前側の幅を `11.6` から段階的に調整し、最終的に `5.8` に変更。
- 縦長ブラウザでステージ手前の左右端が画面外へ切れないようにした。

### 変更理由

- 問題はステージ下部の上下方向の見切れではなく、手前側の左右端が画面外に切れることだったため。
- ステージ手前がカメラに近すぎ、パースで横方向に大きく投影されていたため。

### 確認内容

- `http://127.0.0.1:5500/index_3d.html` を縦長ビューポートで表示し、ステージ左右端が画面内に収まることを確認。

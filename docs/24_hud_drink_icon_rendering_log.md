# HUDドリンクアイコン描画の改善

## 変更日

2026-06-13

## 現状調査

- HUDの `evolution`、`order ticket`、`next` は絵文字とグラデーション背景で表示していた。
- 実際のゲーム内オブジェクトは、グラス、液体、リム、ストロー、氷、フルーツ、ミント、ホイップ、星ピックなどを段階に応じて組み合わせた3Dモデルだった。
- 進化段階の表示は全アイコンが同じ大きさで、オブジェクトのサイズ差が分かりにくかった。

## 変更内容

- HUD用のドリンクアイコンをCanvasで生成するようにした。
- `drinks` の色、半径、高さ、段階ごとの装飾条件を使い、実際のオブジェクトに近い見た目のミニアイコンを描画した。
- `order ticket` と `next` のアイコンも、絵文字ではなく生成したドリンクアイコンに差し替えた。
- `evolution` の各段階アイコンは、ドリンクの半径に応じて表示サイズを変え、サイズ差が見えるようにした。

## 理由

- HUD上のアイコンとステージ上のオブジェクトの見た目を近づけ、プレイヤーが対応関係を理解しやすくするため。
- 進化段階ごとのサイズ差を視覚的に示し、合体後にどれくらい大きくなるかを予測しやすくするため。

## 確認項目

- `evolution`、`order ticket`、`next` のアイコンが絵文字ではなくドリンク形状で表示されること。
- `evolution` の段階が上がるほどアイコンが大きく表示されること。
- 次オブジェクトや注文対象の変更時に、対応するドリンクアイコンが更新されること。

# 3D版シーン管理とゲームモード追加記録

## 2026-06-11

### 変更内容

- `index_3d.html` 単体のまま、タイトル画面、モード選択画面、プレイ画面、結果画面のシーン管理を追加。
- UIレイヤーとして `titleScreen`、`modeScreen`、`resultScreen` を追加。
- ゲーム状態に `scene`、`modeId`、`shots`、`elapsed`、`gameEnded` を追加。
- 以下のモードを追加。
  - 無限スコア: ステージが詰まるまでスコアを伸ばす。
  - 進化チャレンジ: 10段階目を作るまでの射出数を競う。
  - 進化タイムアタック: 10段階目を作るまでのタイムを競う。
  - オーダーラッシュ: 60秒間で注文票の達成とスコアを競う追加候補モード。
- プレイ中の補助表示として、モード、射出数、経過時間を追加。
- プレイ中だけ物理更新、タイマー更新、射出操作が有効になるように変更。
- 10段階目完成時に、進化チャレンジと進化タイムアタックでは結果画面へ遷移するように変更。
- 無限スコアでは、オブジェクト数または手前側の詰まりを検知してゲームオーバーにする簡易条件を追加。
- オーダーラッシュでは、注文達成ボーナスを大きめにし、60秒で結果画面へ遷移するように変更。

### 変更理由

- テスト版のオブジェクト挙動が固まったため、次段階としてスコアやゲーム性を検証できるモード構成が必要になったため。
- HTML完結のまま、今後のゲームルール追加や画面遷移を扱いやすくするため。

### 補足

- 追加候補モードとして、既存の注文票システムと相性がよい「オーダーラッシュ」を実装した。

# 3D版ステージ下部7割レイアウト調整記録

## 2026-06-11

### 変更内容

- `index_3d.html` の3Dキャンバスを画面下部 `70%` に配置。
- 画面上部 `30%` はHUD、スコア、注文票などのUI領域として残す構成に変更。
- 3Dキャンバスの実サイズを使ってカメラアスペクト比とレンダラーサイズを再計算するように変更。
- 下部70%のキャンバス内でもステージが小さくなりすぎないよう、縦長判定のカメラ補正を強めた。
- ステージを下側へ寄せるため、縦長時に投影行列の縦方向オフセットを加えるようにした。

### 変更理由

- 縦長画面でステージの上下方向の占有率が小さく、画面下側に余白が大きく残っていたため。
- ユーザー指定の「画面全体の7割程度をステージにし、残った上部3割でスコア表示などをする」構成に近づけるため。

### 確認内容

- `http://127.0.0.1:5500/index_3d.html` を縦長ビューポートで表示し、ステージが下部70%の領域内に大きく表示されることを確認。

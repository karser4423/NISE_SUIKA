# HUD注文・次オブジェクト表示の短縮

## 変更日

2026-06-13

## 現状調査

- `ORDER TICKET` はラベル、アイコン、「ドリンク名を作る」を表示していた。
- `NEXT` はラベル、アイコン、「n段階目 ドリンク名」を表示していた。
- スマートフォン幅では `order ticket` と `next` が横並びのままでも、文字量が多く折り返しやすかった。

## 変更内容

- `ORDER TICKET` と `NEXT` の項目名は残し、アイコンとドリンク名を短く並べる表示にした。
- `ORDER TICKET` と `NEXT` は、SCOREなどと同じように上段を項目名、下段をアイコンと名称の2段表示にした。
- 注文対象の表示から「を作る」を外した。
- 次オブジェクトの表示から「n段階目」を外した。
- 420px以下で `order ticket` と `next` を縦積みにする指定を外し、短縮後の表示では横並びを維持するようにした。
- 長いドリンク名は枠内で省略表示するようにした。
- 開始メッセージの表示位置を下げ、短縮後のHUD下段と重なりにくくした。

## 理由

- スマートフォン表示で `order ticket` と `next` を1行に収めやすくするため。
- アイコンが実際のオブジェクトに近い見た目になったため、ラベルや段階表記がなくても内容を判別しやすくなったため。

## 確認項目

- `order ticket` 相当の欄がアイコンと名称のみで表示されること。
- `next` 相当の欄がアイコンと名称のみで表示されること。
- スマートフォン幅で `order ticket` と `next` が横並びで表示されること。

# HUD NEXT表示を次の次オブジェクトへ変更

## 変更日

2026-06-13

## 現状調査

- `state.nextLevel` は射出前のプレビューオブジェクトにも使われていた。
- HUDの `NEXT` も `state.nextLevel` を表示していたため、画面上の射出前プレビューと同じオブジェクトを示していた。
- ユーザーには射出前プレビューで現在出るオブジェクトが見えているため、HUDの `NEXT` はその次を表示した方が情報価値が高い。

## 変更内容

- `state.upcomingLevel` を追加し、HUDの `NEXT` は `upcomingLevel` を表示するようにした。
- ゲーム開始時に `nextLevel` と `upcomingLevel` をそれぞれ抽選するようにした。
- 射出時は `upcomingLevel` を次の `nextLevel` に繰り上げ、新しい `upcomingLevel` を抽選するようにした。
- 調整用の「次を変更」ボタンでも `nextLevel` と `upcomingLevel` を両方更新するようにした。

## 理由

- 射出前のステージ上プレビューが現在出るオブジェクトを示しているため。
- HUDの `NEXT` では、現在出るオブジェクトの次に控えているオブジェクトを確認できるようにするため。

## 確認項目

- 射出前プレビューは従来通り現在出るオブジェクトを表示すること。
- HUDの `NEXT` は次の次オブジェクトを表示すること。
- 射出後に、HUDの `NEXT` が次の候補へ更新されること。

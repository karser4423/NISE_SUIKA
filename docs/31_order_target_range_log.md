# 注文対象の抽選範囲拡大

## 変更日

2026-06-13

## 現状調査

- 注文対象は `randomTargetLevel()` で抽選していた。
- 変更前は内部レベル `2` から `6`、表示上の3段階目から7段階目までが対象だった。
- 注文対象をタップ提供できる仕様になり、最大進化オブジェクトも盤面に残るようになったため、上位段階も注文対象にできる状態になった。

## 変更内容

- `randomTargetLevel()` の抽選範囲を内部レベル `2` から `drinks.length - 1` までに変更した。
- 表示上は3段階目から最大10段階目までが注文対象になる。

## 理由

- 7段階目以降のオブジェクトも盤面上で提供できるようになったため。
- 高い進化段階を作って提供する価値を注文システムにも反映するため。

## 確認項目

- 注文対象の抽選範囲が表示上の3段階目から10段階目までになっていること。
- 最大段階も注文対象に含まれること。

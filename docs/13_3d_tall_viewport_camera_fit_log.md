# 3D版縦長画面カメラ調整記録

## 2026-06-11

### 変更内容

- `index_3d.html` に縦長ビューポート向けのカメラ補正処理を追加。
- 画面の横幅に対して高さが長いほど、カメラを少し後ろ・上へ引くようにした。
- 縦長画面では視野角を少し広げ、ステージ手前側が画面下で切れにくくなるようにした。
- リサイズ時に現在の画面比率からカメラ位置、注視点、視野角を再計算するようにした。

### 変更理由

- ブラウザを縦長にしたとき、ステージ下部の手前側が画面外へ切れてしまっていたため。
- スマートフォン確認時に、画面横幅とステージ手前側の見え方が自然に揃うようにするため。

### 補足

- 射出操作は、タッチ開始で位置調整、指を離したタイミングで射出する挙動を維持している。
- 次の射出オブジェクトはタップ前から表示される挙動を維持している。

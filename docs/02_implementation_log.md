# 実装変更記録

## 2026-06-09

### 変更内容

- `01_game_concept.md` の要件に沿って、HTML単体で動作するゲームプロトタイプ `index.html` を追加。
- 南国ビーチのドリンクスタンド風テーマとして、砂浜、海、太陽、パラソル、ヤシの木、木製テーブルを画面に配置。
- 左右位置を調整して正面方向へドリンクを射出する操作を実装。
- ドリンクが板の上を奥へ滑り、後方の行き止まりで停止または押し合う簡易物理を実装。
- 同じ段階のドリンク同士が接触したら、次の段階のドリンクへ合体する処理を実装。
- スコア、コイン、簡易注文票、次ドリンク表示を追加。
- 重力、摩擦、射出速度を画面上のスライダーで調整できるようにした。
- リセットと次ドリンク変更ボタンを追加。

### 変更理由

- UnityでAndroid向けゲームを制作する前に、射出、滑走、停止、合体、注文達成の基本的な面白さをHTMLプロトタイプで確認するため。
- 初期プロトタイプ範囲に合わせ、外部依存なしで短いサイクルで調整できる構成にするため。

### 確認方針

- ブラウザで `index.html` を開き、描画、操作、合体、スコア加算、パラメーター調整が動作することを確認する。

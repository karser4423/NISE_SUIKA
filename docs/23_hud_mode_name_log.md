# HUDモード名表示と射出数モード名称の変更

## 変更日

2026-06-13

## 現状調査

- HUDのMODE表示は `mode.shortName` を使っており、`SCORE`、`SHOT`、`TIME`、`ORDER` の短縮英字が表示されていた。
- モード選択画面の表示名は `mode.name` を使っていた。
- 射出数の少なさを競うモード名は「進化チャレンジ」で、目的が名前から分かりにくかった。

## 変更内容

- HUDのMODE表示を `mode.shortName` から `mode.name` に変更した。
- 「進化チャレンジ」を「効率進化チャレンジ」に変更した。
- READMEのゲームモード表も新名称へ更新した。

## 理由

- プレイ中のHUDでも、モード選択画面と同じ日本語名を表示して分かりやすくするため。
- 射出数を少なくする目的をモード名で伝わりやすくするため。

## 確認項目

- HUDのMODEに「無限スコア」「効率進化チャレンジ」などの日本語名が表示されること。
- モード選択画面に「効率進化チャレンジ」が表示されること。

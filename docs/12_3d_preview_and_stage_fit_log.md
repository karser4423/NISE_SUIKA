# 3D版プレビュー表示とステージ見切れ調整記録

## 2026-06-11

### 変更内容

- タップ前から次の射出ドリンクをステージ手前に表示する挙動へ戻した。
- タッチ開始、スライド、指を離したタイミングで射出する操作は維持。
- ステージ手前側のZ位置をカメラから少し遠ざけ、画面下部で見切れにくくした。
- ステージ手前側の横幅を広げ、スマートフォン画面の横幅に近く見えるよう調整。
- カメラ位置と注視点を調整し、ステージ手前側と画面下部の収まりを改善。

### 変更理由

- 次に射出されるオブジェクトは、タップ前から見えていたほうが狙いを立てやすいため。
- スマートフォン画面下部でステージ手前側が切れており、ステージ幅と画面幅の関係が分かりにくかったため。
